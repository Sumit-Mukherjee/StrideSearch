Defines the sectors for a stride search of polar latitudes. Allocates memory for each sector.

Parameters
pSearchsearch object to be initialized
southernBoundarylatitude of southernmost extent of search domain \( (\ge -90.0 ) \)
northernBoundarylatitude of northernmost extent of search domain \( (\le 90.0 ) \)
sectorRadiusradius of each circular search sector (km)
pslThresholdsectors whose minimum pressures exceed this value will be ignored
vortThresholdsectors whose maximum vorticity falls below this value will be ignored
windThresholdsectors whose maximum windspeed falls below this value will be ignored
coldAirThresholdair-sea temp differential threshold; values above this value will be ignored
vortPslDistThresholdcollocation threshold for location of vorticity max relative to location of psl min
iceThresholdminimum allowable ice fraction per sector
pDatadata container for netcdf input
