Stride Search Conventions
===========

Indices
-------
- 2d indices (i,j): i = latitude index, j = longitude index
- 2d coordinates are (lat,lon), i.e., (lat[i], lon[j])

Units
------
- Spatial units are kilometers
- Angular units are degrees
- Time units are hours

---------
